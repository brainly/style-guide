### Usage

#### Code examples

- basic

```jsx
<Rating />
```

- active with `activeText` & `counterText`

<!-- prettier-ignore -->
```jsx
<Rating
  active
  activeText="Rate this answer"
  counterText="32 votes"
  onChange={onChange}
/>
```

- with a current rate value `rate` and `metricSize`

<!-- prettier-ignore -->
```jsx
<Rating
  rate={2.2}
  metricSize={10}
/>
```

- with a custom label for current value

<!-- prettier-ignore -->
```jsx
<Rating
  rate={2.2}
  aria-label="Our rate:"
/>
```
