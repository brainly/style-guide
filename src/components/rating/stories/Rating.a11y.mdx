### Usage

#### Code examples

- basic

```jsx
<Rating />
```

- with a current rate value `rate`

<!-- prettier-ignore -->
```jsx
<Rating
  rate={2.2}
/>
```

- with a custom label for current value

<!-- prettier-ignore -->
```jsx
<Rating
  rate={2.2}
  ariaRatingLabel="Our rate:"
/>
```
