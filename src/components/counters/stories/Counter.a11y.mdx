### Usage

#### Code examples

- with a label only

<!-- prettier-ignore -->
```jsx
<Counter aria-label="Notifications:">
  32
</Counter>
```

- with an icon

<!-- prettier-ignore -->
```jsx
<Counter aria-label="Points to earn:" icon={ICON_TYPE.POINTS}>
  +15
  <Text type="span" color="text-gray-60" inherited>
    pts
  </Text>
</Counter>
```
