### Rules

| Pattern                                                               | Comment                                                                                                                                             | Status                                      |
| --------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------- |
| **Should** have 4.5:1 contrast ratio                                  |                                                                                                                                                     | Implementation: IN PROGRESS<br />Tests: N/A |
| **Should** have an accessible name for a close button                 | Button can be named by `closeButtonLabel` prop, dafaults to `"close"`.                                                                              | Implementation: DONE<br />Tests: TO DO      |
| **Should** have an accessible name for an icon                        | Icon can be named by `iconTitle` prop, defaults to icon type.                                                                                       | Implementation: DONE<br />Tests: TO DO      |
| **Should** have a possibility to hide an icon from accessibility tree | Icon can be hidden by `iconHidden` prop.                                                                                                            | Implementation: DONE<br />Tests: TO DO      |
| **Should** have an accessible name                                    | Can be named by children, a label specified by `aria-label` prop or a value (`IDREF`) set for the `aria-labelledby` prop that refers to an element. | Implementation: DONE<br />Tests: DONE       |

### Usage

#### Code examples

- with a label only

<!-- prettier-ignore -->
```jsx
<Label aria-label="Your rank">
  Expert
</Label>
```

- with a named icon

<!-- prettier-ignore -->
```jsx
<Label iconType={ICON_TYPE.STAR} iconTitle="Answer rating">
  4/5
</Label>
```

- with a hidden icon

<!-- prettier-ignore -->
```jsx
<Label
  iconType={ICON_TYPE.VERIFIED}
  iconHidden
>
  Answer Verified
</Label>
```

- with a named close button and an icon

<!-- prettier-ignore -->
```jsx
<Label
  buttonLabel="remove attachment"
  onClose={closeLabel}
  iconType={ICON_TYPE.ATTACHMENT}
>
  picture.jpg
</Label>
```
