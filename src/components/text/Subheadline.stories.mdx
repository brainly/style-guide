### Inherited styles

Using prop `inherited` makes Subheadline inherits styles from parent e.g. size. We can still use other props to change appearance.

<Canvas>

          buzzing
        </Subheadline>{' '}
        with the excitement of endless collaboration.
      </Subheadline>
    )}
  </Story>
</Canvas>
