### Rules

| Pattern                                                                                                | Comment                                                                                                                                                                                                                                                                                                                                                                        | Status                                |
| ------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------- |
| **Should** have a proper contrast ratio                                                                | For `medium` size and larger: 3:1 against the background, in other cases: 4.5:1 against the background. <br/> Text color should be chosen to comply with this rule: <br/> - `medium` size text against white background: `indigo-40`, and color variants 50 and darker, <br/> - `medium` size text against black background: color variant 50 and lighter (except `indigo-50`) | Implementation: DONE<br />Tests: N/A  |
| **Should** create a logical content structure                                                          | Logical content structure is created by proper html semantics. Use `type` to set tag.                                                                                                                                                                                                                                                                                          | Implementation: DONE<br />Tests: N/A  |
| **Should** be enlargable                                                                               | The page should be functional when only the text is magnified to 200% of its initial size. User can zoom in and out to change the size of the text. <br/> User default font size is not respected by the component.                                                                                                                                                            | Implementation: DONE<br />Tests: N/A  |
| **Should not** exceed 80 characters in length                                                          |                                                                                                                                                                                                                                                                                                                                                                                | Implementation: N/A<br />Tests: N/A   |
| **Should not** be too small                                                                            | Default document font size (1em) is 16px, `size` prop defaults to `medium`.                                                                                                                                                                                                                                                                                                    | Implementation: DONE<br />Tests: DONE |
| **Should** use semantic markup (ex.`<strong>`, `<em>`) to properly announce style changes to all users |                                                                                                                                                                                                                                                                                                                                                                                | Implementation: DONE<br />Tests: N/A  |
