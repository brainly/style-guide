### Usage

#### Code examples

- nested with strong and emphasis

<!-- prettier-ignore -->
```jsx
<Text>
  Discover <Text type="strong" weight="bold">more</Text> with <Text type="em" color="text-blue-60">Textbook Detective</Text>
</Text>
```

- heading level 1, marked both semantically and visually

```jsx
<Headline type="h1" size="xxxlarge">
  We use cookies
</Headline>
```

- paragraph with a deleted and an inserted part

<!-- prettier-ignore -->
```jsx
<Text type="p">
  Brainly app is so much more than anyone could expected. When someone is just needing help to figure out a <Text type="del" color="text-gray-40" style={{'text-decoration': 'line-through'}}>math</Text> <Text type="ins">chemistry</Text> problem...
</Text>
```

- block quotation as TextBit

<!-- prettier-ignore -->
```jsx
<TextBit type="blockquote">
  This app is so much more than I expected. I was just needing help to figureout a math problem
</TextBit>
```

### Test user settings

```css
* {
  line-height: 1.5 !important;
  letter-spacing: 0.12em !important;
  word-spacing: 0.16em !important;
}

p {
  margin-bottom: 2em !important;
}
```
