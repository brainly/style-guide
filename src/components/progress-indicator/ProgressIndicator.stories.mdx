### With custom theme

To use custom theme for `ProgressIndicator`, you can override following CSS variables:

| Css variable name | Explanation |
| ----------------- | ----------- |
| --barColor        | Bar color   |
| --trackColor      | Track color |

There are two ways of achieving custom theming:

1. Change CSS variables for particular `ProgressIndicator` by defining them inline in the `style` attribute. Example:

```jsx
<ProgressIndicator style={{'--trackColor': 'green'}} />
```

2. Or, you can use `className` and define variables replacements in your CSS file. Example:

```css
.sg-progress-indicator--custom-theme {
  --trackColor: #fbbe2e;
  --barColor: #24a865;
}
```

In the following example, className `sg-progress-indicator--custom-theme` was applied to all `ProgressIndicator` components. The last component has inline variable override:

<Canvas>
  <Story
    name="With custom theme"
    args={{className: 'sg-progress-indicator--custom-theme'}}
  >
    {args => {
      const StyledProgressIndicator = styled.div`
        .sg-progress-indicator--custom-theme {
          --trackColor: #fbbe2e;
          --barColor: #24a865;
        }
      `;
      return (
        <StyledProgressIndicator>
          <ProgressIndicator {...args} />
        </StyledProgressIndicator>
      );
    }}
  </Story>
</Canvas>
