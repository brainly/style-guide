| Pattern                                                                          | Comment                                                                                                                                                                                                                                                                       | Status                                 |

| **Should** have a role `radio`                                                   |                                                                                                                                                                                                                                                                               | Implementation: DONE<br />Tests: TO DO |

| **Should** have an accessible name                                               | Can be named by: <br/>- a label specified by `aria-label` prop<br/>- a value (`IDREF`) set for the `aria-labelledby` prop that refers to an element<br />- can be named by adding children (children should be purely presentational and do not include interactive elements) | Implementation: DONE<br />Tests: TO DO |

| **Should** have visible focus and hover style                                    |                                                                                                                                                                                                                                                                               | Implementation: DONE<br />Tests: N/A   |
| **Should** have a label with 4.5:1 contrast ratio against the background         | `dark` against white: 21:1, `light` against black: 20.9:1                                                                                                                                                                                                                     | Implementation: DONE<br />Tests: N/A   |
| **Should** have a color indicator with 3:1 contrast ratio against the background | `dark` against white: 21:1, `light` against black: 20.9:1                                                                                                                                                                                                                     | Implementation: DONE<br />Tests: N/A   |
| **Should** respect Windows High Contrast mode                                    |                                                                                                                                                                                                                                                                               | Implementation: TO DO<br />Tests: N/A  |

| **Should** respect prefers reduce motion settings                                |                                                                                                                                                                                                                                                                               | Implementation: TO DO<br />Tests: N/A  |

| **Should** be focused and checked by pressing arrows                             |                                                                                                                                                                                                                                                                               | Implementation: DONE<br />Tests: TO DO |

### Usage

#### Code examples

- with a meaningful label

<!-- prettier-ignore -->
```jsx
<Radio
  name="termsservice"
  onChange={checkRadio}
>
  I accept the Brainly Terms of Service & Privacy Policy.
</Radio>
<Radio
  name="termsservice"
  onChange={checkRadio}
>
  I accept only the Brainly Terms of Service.
</Radio>
<Radio
  name="termsservice"
  onChange={checkRadio}
>
  I accept only the Brainly Privacy Policy.
</Radio>
```

- checked

<!-- prettier-ignore -->
```jsx
<Radio
  name="termsservice"
  checked="true"
  onChange={checkRadio}
  required
>
  I accept the Brainly Terms of Service & Privacy Policy.
</Radio>
<Radio
  name="termsservice"
  onChange={checkRadio}
>
  I accept only the Brainly Terms of Service.
</Radio>
<Radio
  name="termsservice"
  onChange={checkRadio}
>
  I accept only the Brainly Privacy Policy.
</Radio>
```

- without a label, but labelled by a custom element

<!-- prettier-ignore -->
```jsx
<h5 id="label-for-radio">
  I accept the Brainly Terms of Service & Privacy Policy.
</h5>
<Radio
  name="termsservice"
  aria-labelledby="label-for-radio"
  onChange={checkRadio}
/>
```

- invalid with description

<!-- prettier-ignore -->
```jsx
<Radio
  name="termsservice"
  invalid="true"
  required="true"
  description="This consent is super important, more on our main page"
  onChange={checkRadio}
>
  I accept the Brainly Terms of Service & Privacy Policy.
</Radio>
<Radio
  name="termsservice"
  onChange={checkRadio}
>
  I accept only the Brainly Privacy Policy.
</Radio>
```
